移除 MIUI 8 的預設瀏覽器

近來凍仁遇到了一個挺令人頭痛的問題，那就是上個月不小心把小米手機 (MI2) 升上 MIUI 8。切換預設瀏覽器的功能癈了，不管設定怎麼改，只要點到開新分頁的連結，都會用 MIUI 內建的瀏覽器開啟！

爬完文的凍仁採用了 root 強制刪除 app 的權宜之計 (workaround)。目前已正常使用一週，或許下次只需移除 com.browser.provider 即可。

▲ MIUI 8 雖未正式 release，但已可從官網取得 MIUI 8 體驗版線刷包。

1. 安裝系統程式安全移除器 (ROOT)。
▲ 於 Google Play 找到系統程式安全移除器 (ROOT) 並安裝。

2. 開啟 MIUI 內建的安全中心。
▲ 開啟桌面上的安全中心。

3. 進入授權管理。
▲ 也可從系統設定進入授權中心。

4. 進入 ROOT 權限管理。
▲ 準備開無雙大絕！

5. 啟用系統程式安全移除器的 ROOT 權限。
▲ 事成之後，還請記得回來關掉 root 權限。

6. 開啟系統程式安全移除器 (ROOT) 應用程式。
▲ 這時的系統程式安全移除器 (ROOT) 已具有 root 權限，請小心使用。

7. 找到瀏覽器、com.browser.provider 並進行刪除。
▲ 剛刪除 App 後，會發現容量剩下 0k，要重開才會根除。

8. 重新啟動手機。

現在大家應該不會再被強制使用 MIUI 8 內建瀏覽器的惡意程式所困擾了！算一算凍仁接手家人換下的 MI2 也快一年了，看來還能再戰幾年呢！

[警告] root 前請三思，可能導致手機無法正常運作！
